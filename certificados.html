Go desde Cero

Go, también conocido como Golang, es un lenguaje de programación de código abierto desarrollado por Google en 2007. En los ultimos años Go se ha posicionado como uno de los leguajes con más rapido crecimiento entre las grandes empresas gracias a su simplicidad, eficiencia y seguridad en su programación concurrente. Además Go utiliza una sintaxis similar a la de C siendo fácil de aprender para los desarrolladores. Go se ha vuelto cada vez más popular en la comunidad de programación debido a su velocidad, escalabilidad y facilidad de uso en entornos de desarrollo modernos.

Programación Orientada a Objetos con Go

Aunque Go es un lenguaje de programación que en principio no se basa en la programación orientada a objetos (POO), Go posee las herramientas para utilizar algunos conceptos de POO para desarrollar programas que sean fáciles de mantener y extender.

Bases de Datos con Go

Go es un lenguaje muy versátil que cuenta con muchas bibliotecas y herramientas para trabajar con bases de datos. Al ser un lenguaje de programación de sistemas, Go se integra muy bien con diferentes tipos de bases de datos, tanto SQL como NoSQL.

Server Side Render con GO

Server Side Rendering (SSR) es una técnica para generar páginas web en el servidor y enviarlas al cliente como HTML completo en lugar de enviar solo un archivo y hacer que el navegador lo renderice. Go puede ser utilizado para implementar SSR utilizando varias herramientas y bibliotecas.
